TABLE

Structured data with bold borders. Information organized clearly.

Example

NameRoleStatus
AmandaDesignerActive
JordanDeveloperActive
CaseyManagerInactive

Usage

import Table, {
  TableHead,
  TableBody,
  TableRow,
  TableCell
} from "@/components/Table";

<Table>
  <TableHead>
    <TableRow>
      <TableCell isHeader>Column 1</TableCell>
      <TableCell isHeader>Column 2</TableCell>
    </TableRow>
  </TableHead>
  <TableBody>
    <TableRow>
      <TableCell>Data 1</TableCell>
      <TableCell>Data 2</TableCell>
    </TableRow>
  </TableBody>
</Table>

Components

ComponentPurpose
TableRoot table wrapper with borders
TableHeadHeader row with black background
TableBodyBody rows with white background
TableRowIndividual row with bottom border
TableCellCell (header or data) with isHeader prop
