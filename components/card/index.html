A design system inspired by record stores, zines, and people who have the right idea but rough edges.

CARD

Square containers with bold borders. Structure that's intentional, not accidental.

Variants

Default

Light borders, clean structure.

Bordered

Bold borders for emphasis.

Elevated

Red shadow for depth.

With Footer

Complete Card

Cards can have headers, content, and footers. Perfect for organizing information with clear hierarchy and actions.

Usage

import Card, {
  CardHeader,
  CardContent,
  CardFooter
} from "@/components/Card";

<Card variant="bordered">
  <CardHeader>
    <h3>Title</h3>
  </CardHeader>
  <CardContent>
    <p>Content</p>
  </CardContent>
  <CardFooter>
    <button>Action</button>
  </CardFooter>
</Card>
