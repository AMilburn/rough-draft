# Rough Draft

Rough Draft is a personal design system focused on a high-contrast, DIY aesthetic. It prioritizes a handmade feel with bold borders and intentional imperfections, inspired by zines and indie record stores.

Live site: [https://AMilburn.github.io/rough-draft/](https://AMilburn.github.io/rough-draft/)

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

To deploy the application to GitHub Pages:

```bash
npm run deploy
```

This command builds the project into the `out/` directory and pushes it to the `gh-pages` branch.
